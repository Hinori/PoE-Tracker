PoE 프로필 링크 관리자 - 개인정보처리방침

본 개인정보처리방침은 'PoE 프로필 링크 관리자' 크롬 확장 프로그램(이하 '확장 프로그램')이 사용자의 정보를 수집하고 사용하는 방식에 대해 설명합니다.

1. 수집하는 정보

확장 프로그램은 다음과 같은 정보를 수집하고 사용자의 컴퓨터에 저장합니다.

  • 사용자가 직접 입력한 Path of Exile 프로필 URL
  • 사용자가 직접 입력한 스트리밍 채널(Twitch, Chzzk, YouTube 등) URL
  • 사용자가 직접 입력한 스트리머 이름
  • 사용자가 지정한 아이콘 정보 (파일 업로드 시 Base64 텍스트, URL 입력 시 해당 URL)

2. 정보의 사용 목적

수집된 모든 정보는 확장 프로그램의 핵심 기능, 즉 사용자가 즐겨찾는 링크 목록을 편리하게 관리하고 접근할 수 있도록 하는 목적으로만 사용됩니다.

3. 정보의 저장 위치 및 기간

모든 정보는 외부 서버로 전송되지 않으며, 오직 사용자의 로컬 컴퓨터 브라우저 내 안전한 저장소(chrome.storage.local API)에만 저장됩니다. 사용자가 확장 프로그램을 삭제하거나 저장된 항목을 직접 삭제할 경우, 해당 정보는 영구히 삭제됩니다.

4. 정보의 제3자 제공

확장 프로그램은 저장된 사용자 정보를 제3자에게 판매하거나 공유하지 않습니다. 단, '라이브 상태 확인' 기능의 정상적인 작동을 위해 아래와 같은 정보가 각 플랫폼의 공식 API 서버로 전송될 수 있습니다.

  • 치지직(Chzzk): 채널명, 채널Icon, 라이브 상태 확인을 위해 채널 ID가 네이버의 치지직 API 서버로 전송됩니다.
  • 유튜브(YouTube): 라이브 상태 확인을 위해 채널 정보 및 개발자의 API 키가 Google의 유튜브 API 서버로 전송됩니다.

위 정보는 해당 스트리밍 채널의 방송 상태를 확인하는 목적 외에는 사용되지 않습니다.
