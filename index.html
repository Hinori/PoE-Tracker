PoE 스트리머 트래커 개인정보처리방침

개인정보처리방침 변경 안내

본 방침은 2025년 9월 22일부터 시행됩니다.

  • 주요 변경 사항: '정보의 제3자 제공 및 공유'(4조) 항목에서 추천 스트리머 목록 및 공지사항을 불러오는 서버의 주소가 명확하게 변경되었습니다.

이전 버전 보기 (2025년 9월 17일자)

시행일: 2025년 9월 22일

본 개인정보처리방침은 'PoE 스트리머 트래커' 크롬 익스텐션(이하 '서비스')이 사용자의 데이터를 어떻게 수집, 사용, 저장 및 보호하는지에 대해 설명합니다. 본 서비스는 사용자의 개인정보를 매우 중요하게 생각하며, 투명하게 운영하는 것을 원칙으로 합니다.

1. 수집하는 정보

본 서비스는 서비스의 핵심 기능 제공을 위해 다음과 같은 정보를 사용자의 명시적인 입력에 의해서만 수집하고 저장합니다. 저희는 사용자의 브라우징 기록, 개인 식별 정보 등을 추적하거나 수집하지 않습니다.

  • 사용자가 직접 입력하는 프로필 정보:
    • Path of Exile 프로필 이름 (게임 태그)
    • 스트리머의 방송 채널 URL (예: Twitch, Chzzk, YouTube)
    • 스트리머 이름 (식별을 위한 메모)
    • 프로필 아이콘 (사용자가 직접 업로드한 이미지 파일 또는 이미지 URL)

위 정보들은 모두 사용자가 서비스의 '관리' 페이지에서 직접 입력하거나 수정할 수 있습니다.

2. 정보의 사용 목적

수집된 정보는 오직 다음의 목적을 위해서만 사용됩니다.

  • 서비스 핵심 기능 제공: 사용자가 등록한 Path of Exile 프로필과 스트리머 채널 목록을 관리하고 팝업창에 표시합니다.
  • 방송 상태 확인: 사용자가 등록한 스트리머 중 '치지직(Chzzk)' 채널의 경우, 실시간 방송 상태(ON/OFF)와 방송 카테고리를 확인하여 팝업창에 표시하기 위해 사용됩니다.
  • 편의 기능 제공: '치지직' 채널 URL 입력 시, 스트리머 이름과 채널 아이콘을 자동으로 완성하여 사용자 편의를 돕습니다.

3. 정보의 저장 및 보안

  • 모든 사용자 데이터는 브라우저의 동기화 스토리지 API(chrome.storage.sync)를 통해 안전하게 저장됩니다.
  • 이 데이터는 사용자가 브라우저에 로그인한 경우, 해당 계정에 연동되어 사용자의 다른 기기에서도 동기화될 수 있습니다.
  • 본 서비스는 별도의 외부 서버나 데이터베이스를 운영하지 않으며, 사용자의 프로필 데이터를 중앙 서버로 전송하거나 저장하지 않습니다. 모든 데이터는 사용자의 브라우저와 계정 내에만 존재합니다.
  • 사용자는 '데이터 내려받기(export)' 기능을 통해 언제든지 자신의 모든 데이터를 JSON 파일 형태로 백업할 수 있습니다.

4. 정보의 제3자 제공 및 공유

본 서비스는 사용자의 동의 없이 개인정보를 제3자에게 판매하거나 공유하지 않습니다. 단, 서비스의 특정 기능을 구현하기 위해 제한적인 정보를 외부 서비스로 전송할 수 있습니다.

  • 네이버 치지직(Naver Chzzk):
    • 사용자가 등록한 '치지직' 채널의 방송 상태를 확인하거나, 관리 페이지에서 URL을 입력하여 채널 정보를 자동 완성할 때, 해당 스트리머의 채널 ID가 네이버의 API 서버(https://api.chzzk.naver.com)로 전송됩니다.
    • 이 과정에서 사용자의 다른 프로필 정보나 개인 식별 정보는 일절 전송되지 않습니다.
  • (수정) GitHub:
    • 서비스 관리 페이지의 '개발자 GitHub에서 목록 업데이트' 기능은 사전에 동의를 받은 추천 스트리머 목록(JSON 파일)을 불러오기 위해 jsdelivr CDN을 통해 GitHub 서버(https://cdn.jsdelivr.net/gh/Hinori/streamer_list.json)에 정보를 요청합니다.
    • 서비스 팝업 페이지의 '공지사항' 기능은 jsdelivr CDN을 통해 GitHub 서버 (https://cdn.jsdelivr.net/gh/Hinori/notice.html)에 정보를 요청합니다.
    • 이 과정에서 사용자의 어떠한 데이터도 GitHub로 전송되지 않습니다. 단순히 공개된 추천 목록을 내려받는 단방향 통신입니다.

5. 사용자의 권리

사용자는 자신의 데이터에 대해 완전한 통제권을 가집니다.

  • 조회, 수정 및 삭제: 사용자는 언제든지 서비스의 '관리' 페이지를 통해 저장된 모든 프로필 정보를 조회, 수정, 삭제할 수 있습니다.
  • 데이터 삭제: 서비스에서 모든 프로필을 삭제하거나 익스텐션을 삭제하면, chrome.storage.sync에 저장된 모든 관련 데이터는 영구적으로 삭제됩니다.
  • 데이터 백업: '데이터 내려받기' 기능을 통해 자신의 데이터를 언제든지 백업할 수 있습니다.

6. 개인정보처리방침의 변경

본 방침의 내용에 추가, 삭제 및 수정이 있을 경우, 업데이트된 내용을 공지하겠습니다.
