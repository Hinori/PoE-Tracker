PoE 스트리머 트래커 개인정보처리방침

시행일: 2025년 9월 12일

본 개인정보처리방침은 Hinori(이하 '개발자')가 개발한 'PoE 스트리머 트래커' 크롬 확장 프로그램(이하 '확장 프로그램')에 적용됩니다.

1. 수집하는 정보

확장 프로그램은 서비스의 핵심 기능을 제공하기 위해 다음과 같은 정보를 사용자의 컴퓨터에 저장합니다. 이러한 정보는 외부 서버로 전송되거나 개발자에게 공유되지 않습니다.

  • 사용자가 직접 입력하는 정보:
    • Path of Exile 계정 이름
    • 스트리밍 채널 URL (Twitch, Chzzk, YouTube 등)
    • 스트리머 이름 (메모)
    • 프로필 아이콘 (이미지 URL 또는 파일 업로드를 통해 Base64로 변환된 데이터)

위 정보는 Chrome의 `storage.sync` API를 통해 사용자의 브라우저에 저장됩니다. 이는 사용자가 구글 계정에 로그인한 경우, 동일한 계정으로 로그인된 다른 기기에서도 목록을 동기화할 수 있도록 합니다.

2. 정보의 사용 목적

수집된 정보는 오직 다음의 목적을 위해서만 사용됩니다.

  • 핵심 기능 제공: 사용자가 저장한 스트리머 목록을 확장 프로그램 팝업 및 관리 페이지에 표시합니다.
  • 바로가기 링크 제공: 저장된 정보를 바탕으로 스트리머의 PoE 프로필 및 방송 채널로 이동할 수 있는 링크를 제공합니다.
  • 실시간 방송 상태 확인: 사용자가 저장한 '치지직' 채널 URL을 바탕으로 치지직 공개 API에 방송 상태를 조회하여, 그 결과를 사용자에게 시각적으로 표시합니다. 이 과정에서 개인 식별 정보는 전송되지 않습니다.
  • 데이터 관리: 사용자가 자신의 데이터를 파일로 내보내거나(백업), 파일로부터 가져올 수 있는(복원) 기능을 제공합니다.

3. 정보의 공유 및 제3자 제공

개발자는 사용자가 저장한 어떠한 정보도 수집, 추적, 또는 제3자에게 판매하거나 공유하지 않습니다. 모든 데이터는 전적으로 사용자의 브라우저 내에 머무릅니다.

단, 확장 프로그램의 특정 기능은 사용자의 요청에 따라 외부 서비스와 통신할 수 있습니다.

  • 치지직(Chzzk): 방송 상태 확인 및 채널 정보 자동 완성을 위해 치지직의 공개 API(`api.chzzk.naver.com`)와 통신합니다.
  • GitHub: 사용자가 '개발자 GitHub에서 목록 업데이트' 기능을 실행할 경우에만, 추천 목록 파일을 받기 위해 `raw.githubusercontent.com`과 통신합니다.

이러한 통신은 공개된 정보(Public Data)를 조회하기 위함이며, 사용자의 개인 프로필 목록이 해당 서버로 전송되는 일은 없습니다.

4. 정보의 보유 및 파기

사용자가 저장한 모든 정보는 사용자가 확장 프로그램을 사용하고 있는 동안 브라우저 저장소에 보관됩니다. 사용자는 언제든지 관리 페이지에서 개별 프로필을 삭제하거나, 확장 프로그램을 삭제함으로써 모든 데이터를 영구적으로 파기할 수 있습니다.

5. 사용자의 권리

사용자는 자신의 데이터에 대한 완전한 통제권을 가집니다. 사용자는 확장 프로그램의 관리 페이지를 통해 언제든지 자신의 데이터를 조회, 수정, 삭제할 수 있습니다.

6. 개인정보처리방침의 변경

본 개인정보처리방침은 법령이나 서비스 변경에 따라 내용이 변경될 수 있습니다. 변경 사항이 발생할 경우, 웹 스토어 게시 페이지나 기타 공지 가능한 방법을 통해 사용자에게 알립니다.

7. 문의하기

본 개인정보처리방침이나 확장 프로그램에 대해 궁금한 점이 있으시면 개발자의 GitHub 페이지를 통해 문의해주시기 바랍니다.
